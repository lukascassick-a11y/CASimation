# Controller Standards

## Supported primary manufacturers

- Honeywell
- Johnson Controls
- Phoenix Controls

## Honeywell Optimizer defaults

| Equipment | Default controller |
|---|---|
| VAV / air valves | `VAA-VA75MB24NMC` |
| Fan coils | `UN-RS0844MS24NMC` |
| Unit ventilators | `UN-RS0844MS24NMC` |
| Heat pumps | `UN-RS0844MS24NMC` |
| AHUs | `UN-RL1644MS24NMC` |
| Boilers | `UN-RL1644MS24NMC` |
| Chillers | `UN-RL1644MS24NMC` |
| Pump systems | `UN-RL1644MS24NMC` |

## Johnson Controls standards

- VAV default: `F4-CVM3050-0`
- Other-equipment default: `F4-CGM09090-0`
- `FX-` products are legacy and hidden from normal new-estimate lists.

## Exclusions

- Siemens controllers are not selectable.
- Removed products never appear in estimating lists.
