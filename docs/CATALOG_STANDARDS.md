# Catalog Standards

## Product lifecycle

- **Active** — selectable for new estimates.
- **Legacy** — hidden by default but available for historical work when enabled.
- **Discontinued** — retained for historical records but unavailable for new estimates.
- **Removed** — excluded from all program selection lists.

## Naming

- Use `Honeywell`, `Johnson Controls`, and `Phoenix Controls` as canonical manufacturer names.
- Use `Optimizer` as the canonical spelling.
- Stable part numbers should be used as internal identifiers whenever possible.

## Proprietary source workbook

The legacy workbook is intentionally ignored by Git. Keep it locally in a `reference_workbooks/` folder when formula comparison or data extraction is required.
